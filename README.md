# testWelbeX

## backend
В файле `testWelbeX/backend/routes/main.js` поменять данные для подключения к postgres.

Для запуска сервера:

```
cd ./testWelbeX/backend
npm i
npm start
```

## frontend
Для запуска приложения:

```
cd ./testWelbeX/frontend
npm i
npm start
```
